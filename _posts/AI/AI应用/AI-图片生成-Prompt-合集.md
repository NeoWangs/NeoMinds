## 等距微缩模型场景

**适用场景：** 将任意建筑生成为精致的等距微缩模型风格图片，可搭配不同天气效果

**模板：**

```
以清晰的 45° 俯视角度，展示一个等距微缩模型场景，内容为 [建筑]，
天气效果巧妙融入场景中，柔和的 [天气效果] 与建筑轻柔互动。
使用基于物理的真实渲染（PBR）和逼真的光照效果，纯色背景，清晰简洁。
画面采用居中构图，凸显出三维模型精准而细腻的美感。
在图片上方展示建筑名称。
```

**变量说明：**
- `[建筑]`：目标建筑，可以是具体地标或附图建筑
- `[天气效果]`：如下雪、雨后、晨雾、黄昏、樱花飘落等

**示例一：径山寺·雪景**

```
以清晰的 45° 俯视角度，展示一个等距微缩模型场景，内容为径山寺，
天气效果巧妙融入场景中，柔和的下雪效果与建筑轻柔互动。
使用基于物理的真实渲染（PBR）和逼真的光照效果，纯色背景，清晰简洁。
画面采用居中构图，凸显出三维模型精准而细腻的美感。
在图片上方展示建筑名称。
```

## Q 版角色表情包（LINE 贴图风格）

**适用场景：** 将参考图中的角色生成为 LINE 风格 Q 版半身像贴图合集，适合用于聊天表情包制作

**使用方式：** 上传参考角色图 + 粘贴以下 prompt

```
为参考图中角色绘制 Q 版的、LINE 风格的半身像表情包，注意头饰要正确还原。
彩色手绘风格，使用 4×6 布局（共 24 格），涵盖各种各样的常用聊天语句，
或是一些相关的娱乐 meme。
不要直接复制原图，需进行 Q 版风格化再创作。
所有文字标注为手写简体中文。
生成图片为 4K 分辨率，16:9 比例。
```

**注意事项：**
- `头饰要正确`：上传图需清晰展示头饰细节，prompt 中可补充描述（如"戴白色发带"）
- `4×6 布局`：共 24 个表情格，若角色表情不够丰富可适当减少至 3×6 = 18 格
- `手写简体中文`：每格配一句话，如"好的好的""累了""哈哈哈""在吗""不想动"等

## 折纸艺术

**适用场景：** 将任意主体生成为折纸风格写实插图，背景可搭配具体场景

**模板：**

```
折纸艺术，真实纸张质感，几何折痕，细节丰富，线条锐利，对称构图，
【[主体]】，颜色丰富栩栩如生，【[背景]】背景，柔和自然光，电影级景深效果。
```

**变量说明：**
- `[主体]`：折纸的主体形象，如动物、建筑、神话生物等
- `[背景]`：衬托主体的场景，如森林、西湖、星空、雪山等

## 折页风涂鸦插画（Decorative Folk Flat Illustration）

**适用场景：** 将任意图片转化为民俗风格的扁平涂鸦插画，色彩大胆活泼，呈现手绘童趣感（图生图模式）

**使用方式：** 上传原图 + 粘贴以下 prompt

```
Please transform the entire image into a single Decorative Folk Flat Illustration with Doodle elements. Use a bold and playful color palette, completely different from the original image. Simplify all details into clean, flat shapes with a handmade, slightly imperfect feel, as if drawn on a sheet of white paper. The overall style should look cute, childlike, and whimsical.
```

**效果说明：**
- 扁平化处理，简化所有细节为干净的几何色块
- 配色与原图完全不同，大胆活泼
- 手绘感、略带不完美的线条，呈现在白纸上的涂鸦质感
- 整体风格可爱、童趣、奇幻
