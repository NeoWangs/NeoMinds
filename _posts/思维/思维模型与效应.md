## 一、马斯洛需求层次理论

人类的需求从低到高分为五层，低层需求满足后才会追求高层：

```
        /  自我实现  \        ← 成为最好的自己
       /    尊  重    \       ← 被认可、有成就感
      /    情  感      \      ← 归属感、爱与被爱
     /     安  全       \     ← 人身、财产、健康
    /      生  理        \    ← 吃饭、喝水、睡觉
```

> [!tip] 对下属/团队的管理启示
> 满足四个层次：**温饱、尊重、希望、自我实现**。

## 二、巴纳姆效应（Barnum Effect）

人很容易受到来自外界信息的暗示，从而出现自我知觉的偏差——认为一种笼统的、一般性的人格描述十分准确地揭示了自己的特点。

> [!example]
> 星座运势说"你外表坚强但内心敏感"，几乎每个人都会觉得说的就是自己。

### 如何避免，客观认识自己？

1. **学会面对自己**——别因为缺陷或自认为的缺陷，就用自己的方式掩盖
2. **培养收集信息的能力和敏锐的判断力**
3. **以人为镜**——选择条件相当的人做比较（爱因斯坦家邻居杰克大叔的故事）
4. **通过重大事件认识自己**——特别是重大的成功和失败

## 三、自我实现的预言（Self-Fulfilling Prophecy）

当你预言某事会发生后，你会有意无意地推动事情朝预言的方向发展，于是预言成真。

> [!example]
> 老师认为某个学生聪明，就会给他更多关注和鼓励，学生因此表现更好，验证了老师的"预言"（即罗森塔尔效应/皮格马利翁效应）。

## 四、荷塘效应（复利思维）

> 浮萍面积每天扩大一倍，30 天涨满池塘，多少天占到一半的面积？
> 答案是**第 29 天晚上**。

启示：指数增长的前期看起来微不足道，但临界点之后会爆发式增长。做事要耐得住前期的缓慢积累。

---

## 五、合成谬误（Fallacy of Composition）

认为**对个体正确的事情，对集体也一定正确**。

> [!example]
> 一个人在剧院里站起来可以看得更清楚，但如果所有人都站起来，谁也不会看得更清楚——反而所有人都更累了。

---

## 六、囚徒困境（Prisoner's Dilemma）

在个人利益驱动下做出的各自"最优"策略组合，产生了一个**更差的总体收益**。

| | 囚徒 B 沉默 | 囚徒 B 背叛 |
|---|-----------|-----------|
| **囚徒 A 沉默** | 各判 1 年（最优总体） | A 判 10 年，B 释放 |
| **囚徒 A 背叛** | A 释放，B 判 10 年 | 各判 5 年（实际结果） |

每个人从自身利益出发都会选择背叛，最终双方都判 5 年——比合作的结果差得多。

## 七、奶头乐理论（Tittytainment）

由于生产力的不断上升，世界上一大部分人口将不用也无法积极参与产品和服务的生产。为了安慰这些"被遗弃"的人，他们的生活应该被大量的娱乐活动填满——网络、电视、游戏等。

> [!warning]
> 该理论常被用来解释为什么短视频、手游等低门槛娱乐如此流行，以及为什么"奶头乐"可能会消磨人的进取心和深度思考能力。

## 八、四象限法则

### 时间管理四象限（艾森豪威尔矩阵）

|  | **紧急** | **不紧急** |
|---|---------|-----------|
| **重要** | 第一象限：**优先解决** | 第二象限：**计划去做** |
| **不重要** | 第四象限：**有空再说** | 第三象限：**给别人做** |

> [!tip] 核心原则
> 大部分人的时间被第一象限（救火）和第四象限（琐事）吞噬。真正拉开差距的是**第二象限**——重要但不紧急的事（学习、健康、规划），越多投入第二象限，第一象限的"救火"就越少。

### 认知四象限（Johari Window 变体）

|  | **Know（知道）** | **Unknown（不知道）** |
|---|-----------------|---------------------|
| **Know（知道）** | Know-Know 知道自己知道（看山还是山） | Know-Unknown 知道自己不知道（看山不是山） |
| **Unknown（不知道）** | Unknown-Know 不知道自己知道 | Unknown-Unknown 不知道自己不知道（看山是山） |

### 协作心态四象限

|  | **集体主义** | **个人主义** |
|---|------------|------------|
| **开放心态** | **合作**（第一象限） | **同流**（第四象限） |
| **封闭心态** | **竞争**（第二象限） | **冲突**（第三象限） |

### 四象限综合映射

|  | **紧急** | **不紧急** |
|---|---------|-----------|
| **重要** | **第一象限：优先解决** | **第二象限：计划去做** |
|  | Know-Know（看山还是山） | Unknown-Know |
|  | 合作（开放 + 集体） | 竞争（封闭 + 集体） |
| **不重要** | **第四象限：有空再说** | **第三象限：给别人做** |
|  | Know-Unknown（看山不是山） | Unknown-Unknown（看山是山） |
|  | 同流（开放 + 个人） | 冲突（封闭 + 个人） |

## 九、情绪价值与情绪成熟

### 情绪价值

$$\text{价值} = \text{收益} - \text{成本}$$

在亲密关系中，一个人能否为伴侣提供**情绪价值**——即伴侣所感知到的情绪受益大于情绪成本，很大程度上可以决定一段关系的成败。

### 情绪成熟（Emotional Maturity）

能根据不同情境，做出适当、积极的反应。情绪成熟的人有三个特点：

| 特点 | 说明 |
|------|------|
| **负责** | 知道为自己的情绪负责，也愿意为自己给他人造成的负面情绪负责。不会寄希望于对方完全为自己的情绪负责 |
| **有适应能力** | 有能力根据场景变化调节自己的情绪，控制自己在情绪状态中的行为反应 |
| **给予** | 一种付出的能力，在情感上不只关注自己的期望和要求，也会考虑他人、同理他人感受 |

### 四个实操方法

1. **积极建设性回应**（Active-constructive Responding）— 对方分享好消息时，用热情和追问来放大对方的喜悦，而不是敷衍或泼冷水
2. **对日常小事表达高质量的感恩** — 不是泛泛地说"谢谢"，而是具体说出对方做了什么、你的感受是什么
3. **秘密地为压力大的伴侣提供支持** — 不邀功、不提醒"我为你做了多少"，默默减轻对方的负担
4. **软化你开启对话的方式**（Soften Your Start-up）— 在争论爆发前，用温和而非攻击性的方式开启对话

## 十、敏捷开发四大价值观

来自《敏捷宣言》（Agile Manifesto，2001）：

| 更有价值的（左） | 胜过 | 也有价值但次要的（右） |
|-----------------|------|---------------------|
| **个体及交互** | > | 流程与工具 |
| **可用的软件** | > | 冗长的文档 |
| **与客户的协作** | > | 合同谈判 |
| **对变化的响应** | > | 遵循计划 |

> [!info]
> 右边的也有价值，但在需要取舍时，左边的优先级更高。
