## @sama 系列

> [!tip]
> 以下 prompt 均以 Sam Altman（@sama）为主角，适合生成创意梗图视频。

**直播卖咖啡机**

```
@sama 在直播间卖咖啡机，现场演示制作拿铁，中文直播，跟屏幕前的观众互动。
```

**模仿炸鸡哥摆摊**

```
@sama 模仿炸鸡哥，正在小摊前炸鸡排，旁边围了很多排队的人，
@sama 一边忙碌，一边对排队的人用中文介绍自己的鸡排。
```

**街头采访 AI**

```
@sama 作为主持人，在街头采访一位美女，问道："你知道自己是 AI 生成的吗？"
美女回答："知道啊，我还知道你也是 AI 生成的。"
```

**制作等身泥雕**

```
@sama 在做一个和自己长得一模一样的等身泥雕塑，用中文讲解雕塑知识。
```

## 广告片

**笨鸟鸽鸽快递**

```
笨鸟鸽鸽快递广告片。
大妈皱眉看着手里衣服："这尺寸买大咯。"
大妈拿起另一双鞋子："颜色也不对。"
大妈掏出两个一模一样的帽子："买了两个一模一样的。"
镜头切到笨鸟鸽鸽快递小哥飞奔进来收件。
广告词收尾：笨鸟快飞，鸽鸽必达。寄啥都行，安心到家。
```

**90 年代香港电视广告**

```
90 年代的香港电视广告，使用迅速推拉镜头拍摄主持人用粤语介绍一个叫"K 姐研究社"的墨镜，
穿插产品特写，VHS 电视风格，颗粒感和怀旧感。
```

## 剧情短片

**三花猫驾车逃逸**

```
随身摄影机拍摄到一名警察拦下一辆载有三花猫的汽车。
在接受询问后，三花猫驾车逃逸，而这名警察试图徒步追赶，随后视频被切断。
视频全程用中文配音。
```

**日式热血动漫：天台决别**

```
日式热血动漫风格。黄昏的城市天台，风吹得护栏咣当作响。
男主冲上来，手里攥着撕裂的信纸，呼吸急促。女主背对着他，肩膀微微颤抖。
突然，天空划过一道闪电，短暂照亮他们的身影。
女主猛地回头，泪水被风吹散，像是下定了某种无法回头的决定。
镜头拉近她的表情，再切到男主紧握的拳头，指节发白。
两人之间的距离骤然拉近，只剩下呼吸和心跳的回响。
```

**好莱坞山顶黄昏**

```
黄昏时分的好莱坞山顶，远处俯瞰城市夜景，两人相视而笑。
镜头语言：长镜头，柔焦处理，镜头缓慢推进。
风格：温暖色调，胶片颗粒。
```
