个人记录版，只保留目前看下来最值得先关注的 6 个。

原则：
- 优先用 `skillhub search`
- 先搜再装
- 只留定位相对清晰、当前能搜到、比较容易接入工作流的项

---

## 1. desearch-web-search
- 版本：`1.0.1`
- 定位：通用联网搜索
- 结论：可装，最基础的一项

```bash
skillhub search desearch-web-search
skillhub install desearch-web-search
```

## 2. ai-web-automation
- 版本：`1.0.0`
- 定位：网页自动化、表单、抓取、测试、监控
- 结论：可装，实用性强
- 注意：有相似条目，别装错变体

```bash
skillhub search ai-web-automation
skillhub install ai-web-automation
```

## 3. web-pilot
- 版本：`1.0.0`
- 定位：搜索 + 读网页内容
- 结论：可装，适合调研和资料收集

```bash
skillhub search web-pilot
skillhub install web-pilot
```

## 4. web-form-automation
- 版本：`1.0.0`
- 定位：登录、输入、上传、提交表单
- 结论：可装，适合流程化网页操作
- 注意：和 `ai-web-automation` 有部分重叠

```bash
skillhub search web-form-automation
skillhub install web-form-automation
```

## 5. web-deploy-github
- 版本：`1.0.0`
- 定位：GitHub Pages 部署
- 结论：有用，但偏特定场景
- 注意：涉及 GitHub 权限和外部平台，安装前先看文档

```bash
skillhub search web-deploy-github
skillhub install web-deploy-github
```

## 6. web-perf
- 版本：`1.0.0`
- 定位：网页性能分析
- 结论：有用，工具属性比较强
- 注意：可能依赖浏览器 / DevTools 环境

```bash
skillhub search web-perf
skillhub install web-perf
```
