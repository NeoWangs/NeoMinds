## 一、常用前缀

### 1. un-（不，非，无）
**un + important** → (不)(重要的) → 不重要的

**un + able** → (不)(.adj) → 不能的，不会的

### 2. in-（向内，进入）
**in + crease** → (向内)(增长) → 增加

**in + cid + ent** → (进入)(降临)(.n) → 事件

### 3. de-（向下，加强，使）
**de + crease** → (向下)(增长) → 减少

**de + termin + e** → (加强)(界限) → 下决心

**de + termin + ation** → (表加强，使)(界限) → 决定

### 4. pre-（预先，以前）
**pre + fer** → (预先)(带来) → 更喜欢，宁愿

**pre + vi + ous** → (以前)(路，走)(...的) → 以前的，早先的

### 5. re-（再，重复）
**re + semble** → (再)(相似性) → 类似

### 6. be-（使，介词）
**be + fore** → (介词)(前面) → 在...以前；在...前面

**be + hind** → (介词)(藏在后面) → 在...后面；在原处；屁股

**be + long** → (使)(长) → 属于，适应

### 7. en-（进入...中，使）
**en + velop + e** → (进入...中)(=cover盖上) → 信封，封套

**en + joy + able** → (进入...中)(愉快)(.adj) → 令人愉快的

### 8. ac-（一再，给予，加强）
**ac + cess** → (一再)(go行走) → 通道，入口，接近

**ac + quaint + ance** → (一再)(告知)(.n) → 熟人

**ac + know + ledge** → (给予)(知道)(突出部分) → 承认...的权威，对...打招呼

### 9. ap-（一再，加强）
**ap + point + ment** → (加强)(点)(.n) → 约会，预约

**ap + proach** → (一再)(接近) → v接近，n途径

**ap + pear** → (一再)(看见) → 出现，显露

**ap + par + ent** → (=pear)(…的) → 显然的

**dis + ap + pear** → (不，非，无) → 消失

### 10. a-（表加强）
**a + spect** → (表加强)(看，看的东西) → 外表，容貌

### 11. ar-（一再）
**ar + rive** → (一再)(河) → 到达，达成

### 12. con-（共同，全部，一起）
**con + cept** → (共同)(掌握) → 概念，设计

**con + crete** → (共同)(=cre制造) → 混凝土

**con + tin + ent** → (共同)(握)(表物) → 大陆

**con + sid + er** → (一起)(坐)(.n) → 思考

**con + sid + er + ate** → (...的) → 细心的

**con + tain + er** → (全部)(握住)(人/物) → 容器，集装箱

### 13. oc-（再次）
**oc + cup + ation** → (再次)(拿)(.n) → 占领，职业

### 14. over-（在上/外，超过）
**over + come** → (在上/外)(来) → 战胜，克服

**over + head** → (在上/外)(头) → 在头顶上的，架空的

### 15. di-（分开）
**di + stant** → (分开)(站立) → 遥远的

**di + rect + ion** → (使..变)(直)(.n) → 方向，指令

### 16. ex-（出）
**ex + ist + ence** → (出)(站立)(.n) → 存在

### 17. im-（不，无，非）
**im + possible** → (不，无，非)(可能的) → 不可能的

### 18. sur-（超过，在...上面）
**sur + face** → (超过，在..上面)(脸) → n表面，v显露

### 19. under-（在...下）
**under + ground** → (在...下)(地下) → 地铁，秘密的

### 20. through-（通过）
**through + out** → (通过)(在外部) → 遍布，到处

### 21. per-（全部）
**per + hap + s** → (全部)(机会/运气)(复数) → 也许，可能

### 22. pro-（在前面）
**pro + fess + ion** → (在前面)(说)(.n) → 职业

**pro + fess + or** → (人) → 教授

## 二、常用后缀

### 1. -ent（...的/表人）
**anci + ent** → (古老)(...的) → 远古的

**sci + ent + ific** → (知道)(...的)(adj) → 科学的

**pati + ent** → (病)(表人) → 病人

### 2. -able（能...的，.adj）
**avail + able** → (利用)(能..的 .adj) → 可利用的

**prob + able** → (测试，证明)(.adj) → 可能的，大概的

**enjoy + able** → (愉快)(.adj) → 令人愉快的

### 3. -er（人/物，前者）
**form + er** → (=fore先前)(人/物) → 前者，以前的

**stran + ger** → (陌生的) → 陌生人

**contain + er** → (人/物) → 容器，集装箱

### 4. -ive, -ist, -ian（表人）
**relat + ive** → (联系)(表人) → 亲戚

**dent + ist** → (牙齿)(人) → 牙医

**christ + ian** → (基督)(人) → 基督教徒

### 5. -ion（.n）
**direct + ion** → (直)(.n) → 方向，指令

**pos + sess + ion** → (能)(坐)(.n) → 所有物，财产

**product + ion** → 生产

**fict + ion** → (塑造)(.n) → 小说

**un + ion** → (单一)(.n) → 联合，工会

**profess + ion** → (说)(.n) → 职业

**occup + ation** → (拿)(.n) → 占领，职业

**popul + ation** → (人民)(行为/状态) → 人口

### 6. -el（人/物）
**nov + el** → (新的)(人/物) → 小说

### 7. -ar（.n/...的）
**coll + ar** → (脖子)(.n) → 衣领

**simil + ar** → (一样)(...的) → 同样的

**famili + ar** → (家庭)(...的) → 熟悉的

### 8. -ary（.n）
**sal + ary** → (盐)(.n) → 工资

**prim + ary** → (第一，主要)(.adj) → 首选的

### 9. -ure（n）
**cre + at + ure** → (产生)(=ate)(n) → 创造物，生物

### 10. -ty（.n）
**plen + ty** → (满)(.n) → 大量，丰富

**socie + ty** → (社会)(.n) → 社团

### 11. -y（抽象n）
**theor + y** → (思考)(抽象n) → 理论

### 12. -th（抽象名词）
**streng + th** → (强壮的)(抽象名词) → 力量

**dep + th** → 深度，深奥

**grow + th** → 成长

### 13. -ance/-ence（.n状态，行为）
**acquaint + ance** → (告知)(.n) → 熟人

**enter + ance** → (进入)(.n状态，行为) → 入口

**exist + ence** → (站立)(.n) → 存在

### 14. -ice（行为状态）
**pract + ice** → (实践)(行为状态) → 实践

### 15. -ative（有...作用的）
**altern + ative** → (轮流的)(有..作用的) → 替代物，选择

### 16. -al（...的）
**speci + al** → (种类)(…的) → 特别的

### 17. -ly（表方式，行为，状态，.adv）
**eventual + ly** → (最后的)(表方式，行为，状态) → 终于，最后

**probab + ly** → (=prob+able)(.adv) → 可能地

**deep + ly** → 深深地

### 18. -ward(s)（向...）
**after + ward + s** → (随后)(向...后) → 后来，然后

### 19. -ship（.n）
**owner + ship** → (所有者)(.n) → 所有权

### 20. -dom（表状态/领域）
**wis + dom** → (=wise智慧的)(表状态/领域) → 明智，智慧

### 21. -ing（.n）
**strock + ing** → (筒状的)(.n) → 长筒袜

### 22. -ness（.n）
**will + ing + ness** → (心愿)(.n)(.n) → 主动，自愿

**sick + ness** → 疾病

### 23. -edy（.n）
**com + edy** → (宴会)(.n) → 喜剧

### 24. -atic（...的）
**system + atic** → (系统)(..的) → 有系统的

### 25. -ibly（...地）
**poss + ibly** → (可能)(...地) → 可能地，也许

### 26. -cate（.n）
**cert + ifi + cate** → (搞清)(=fy表动)(.n) → 证件

### 27. -ment（.n）
**appoint + ment** → (点)(.n) → 约会，预约
